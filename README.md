## Simple API Handler In *C* For SQLite

### Installation
```bash
git clone --depth=1 https://github.com/amritxyz/api_handler.git
cd api_handler
make run
```

### Dependencies

- SQLite3 \
- [Clang](https://clang.llvm.org/) or [GNU compiler](https://gcc.gnu.org/) \
